# Superstore Sales Analysis

**Contents**
- `superstore_sales.csv` (~5000 rows) - synthetic sample dataset
- `queries_step_by_step.sql` - 10 step-by-step SQL queries with purpose/explanation
- `superstore_analysis.py` - Pandas script: cleaning, aggregations, outputs, and charts
- Output files generated by the script: monthly_sales_summary.csv, region_performance.csv, category_performance.csv, subcat_performance.csv, top_customers.csv, loss_orders.csv, monthly_sales_trend.png, category_profit.png

## How to run (VS Code / Terminal)

1. Extract the ZIP and open the folder `Superstore_Sales_Analysis` in VS Code.
2. Open terminal (Ctrl+`).
3. Run the analysis script:
   ```bash
   python superstore_analysis.py
   ```
4. The script will create summary CSVs and PNG charts in the same folder.

## SQL Queries (Short guide)
- Use `queries_step_by_step.sql` in a MySQL or compatible environment after loading `superstore_sales.csv` into a table named `superstore_sales`.
- Each query is numbered and includes a short purpose/explanation as comments.
